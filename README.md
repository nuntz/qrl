# QRL

QRL is a dependency-free, single-file QR code generator for text and URLs.

## Usage

Open `index.html` in a browser, type or paste text into the input, and the QR code updates in real time.

## Notes

- This project was vibe-coded.
- Byte mode (UTF-8) only.
- Error correction level is fixed to Medium (M).
- Output uses a fixed module size with a 4-module quiet zone.
- No external libraries.
